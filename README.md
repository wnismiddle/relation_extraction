# REST API

URL 结构 http://127.0.0.1:5000/method


### 通用返回结果
```python
# 正常返回结果
{
    "ret_code" : 0,          # 返回码，0为ok
    "task" : "relation_ex"      # 输入的任务
}
```

```python
# 异常返回结果
{
    "ret_code" : -1,         # 返回码，非0表示错误
    "task" : "relation_ex",     # 输入的任务
    "err_info" : "..."      # 具体错误信息
}
```
